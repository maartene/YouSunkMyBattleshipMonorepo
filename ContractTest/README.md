# ContractTest
Small cli utility that can drive the YouSunkMyBattleship backend for testing purposes.

## Building
```bash
# swift build --configuration release
```

## Running
```bash
# .build/release/ContractTest <hostname> <port>
```
ℹ️ Default hostname is `localhost` and default port is `8080`. Either pass no arguments, one argument (hostname) or two arguments (hostname and port).
